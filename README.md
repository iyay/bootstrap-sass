# Bootstrap for Sass

This is Ice House for of [bootstrap-sass](https://github.com/thomas-mcdonald/bootstrap-sass) using the latest version (2.2.1) of [Twitter Bootsrap](http://twitter.github.com/bootstrap/)

## Usage

### Rails

In your Gemfile:

    gem 'sass-rails', '~> 3.1'
    gem 'bootstrap-sass', :git => 'git://github.com/iyay/bootstrap-sass.git'
